## How to Add Images

1. **Using Finder (Mac):**
   - Open Finder
   - Press `Cmd + Shift + G` (Go to Folder)
   - Paste: `/Users/ramjhawar/Downloads/OSKI_LIFTS/OSKILIFTS/src/assets/oski`
   - Press Enter
   - Drag and drop your Oski images here

2. **Using Terminal:**
   ```bash
   cd /Users/ramjhawar/Downloads/OSKI_LIFTS/OSKILIFTS/src/assets/oski
   # Then copy your images here
   ```

3. **Using VS Code/Cursor:**
   - Right-click on the `src` folder in the file explorer
   - Select "New Folder" → name it `assets`
   - Right-click on `assets` → "New Folder" → name it `oski`
   - Right-click on `oski` → "Reveal in Finder" (or "Open in File Manager")
   - Drag your images into that folder

## Image Requirements

- **File Name:** `oski-base.png` (or `oski-base.jpg`) for the main image
- **Optional Variants:** `oski-variant1.png`, `oski-variant2.png`, etc.
- **Format:** PNG (preferred, supports transparency) or JPG
- **Size:** 200x200px to 400x400px recommended
- **Background:** Transparent background preferred
- **Content:** Oski should be centered in the image
